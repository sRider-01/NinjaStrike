# Ninja Strike

## 概要

C++ / DxLib で制作した 2D 忍者アクションゲームです。
「遊んだあとに誰かに話したくなる体験」を設計の軸として開発しました。

## ゲーム内容

### ステージ構成

チュートリアル : 踏んだ場所でヒントが表示される操作説明ステージ

ステージ1 : 雑魚敵・遠距離敵と戦いながらボスへ向かう

ボス戦（通常） : ランダム行動のボスと対決

ボス戦（強化） : 隠しルートから挑む強化形態のボス

### 隠し要素

ステージ１の画面右端のタイルが欠けている箇所より手前の範囲でボスマップに遷移すると強化ボスへのルートがある

---

## 技術的なアピールポイント

入力レイテンシ計測 : 0.05秒以内を数値目標に設定・画面表示で可視化

当たり判定 : 矩形・円・円×矩形の3種をCollisionBaseに集約

オブジェクトプール : エフェクトをstd::arrayで静的確保・再利用

パラメータ駆動 : BossParameterで通常/強化形態を切り替え

delta_time基準 : フレームレート非依存のアニメーション管理

TryGetパターン : bool+out引数でnullチェック不要な統一設計

カメラ2モード : 線形補間追従とスナップ瞬間移動を使い分け

コーディング規約 : auto不使用・全const・全関数XMLコメント統一
